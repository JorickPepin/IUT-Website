Apprendre le français

À l'IUT, la majorité des classes sont offertes en français.

Pour améliorer vos aptitudes, vous pouvez prendre des cours supplémentaires avec l'université de Bourgogne. Pour plus d'informations, consultez le site du CIEF (Centre International d'Etudes Françaises).

Candidature

En tant qu'étudiant étranger, vous avez 3 possibilités pour venir étudier :

  • À titre individuel :

    Si vous ne faites pas partie du programme Erasmus et que votre université n'a pas de partenariat avec l'IUT, veuillez suivre les instructions affichées sur la page de l'uB pour les étudiants venant à titre individuel.

  • Dans le cadre d'Erasmus :

    Si vous faites partie du programme Erasmus, veuillez suivre les instructions à cet effet sur la page de l'uB pour les étudiants Erasmus.

  • Dans le cadre d'un partenariat avec votre université d'origine :

    Si votre université a un partenariat avec l'IUT, veuillez suivre les instructions à cet effet sur la page de l'uB pour les échanges étudiants. Pour vérifier si votre université est partenaire avec l'IUT, vous pouvez consulter la page Partir à l'étranger ou vous rendre directement sur le site de l'IUT.
